# Team Task Manager

A full-stack Team Task Manager web application where users can create projects, assign tasks, and track progress with role-based authentication.

---

## Features

- User Signup & Login
- JWT Authentication
- Role-Based Access (Admin / Member)
- Create Projects
- Create Tasks
- Task Status Tracking
- Dashboard with:
  - Total Tasks
  - Overdue Tasks
- MongoDB Database
- REST APIs
- Fully Responsive Frontend

---

## Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

### Deployment
